COVID-19 Trinidad and Tobago API

This page describes the COVID-19 Trinidad and Tobago API.

Objective

The API described here is an attempt to make the statistical data being provided by the Trinidad and Tobago Ministry of Health (MoH) about the impact of the COVID-19 virus on Trinidad and Tobago publicly available in a form that developers build upon. The hope is that multiple views of the data can be generated by developers now that this dataset is available in an API form.

Source

The source of the data are the media releases from the MoH. From these public media releases tabular data is extracted and made available in a structured form that can be built on.

Usage

This API consists of a single endpoint, 

/updates
. Each update represents the tabular data from the MoH press release.

All updates

Endpoint

https://us-central1-rona-data-tt.cloudfunctions.net/updates

Query parameters

Query parameters from and to can be used to restrict responses to updates with a timestamp >= the from value and/or <= the to value respectively.
